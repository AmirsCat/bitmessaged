Gateway page for mobile bitmessage applications

This page is a free service.
It is still in development, but will provide functions so mobile devices (smart phones) can share the bitmessage network.
The service is implemented, using the opene source bitmessage daemon .

  • gateway overview
  • known bitmessage nodes
  • current bitmessage objects
  • Post an object (this function is experimental, use only for tests!).
